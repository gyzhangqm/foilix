XFOIL est un programme interactif pour la conception et l’analyse des profils subsoniques isolés.
Il se compose d’un ensemble de routines et de leurs menus qui exécutent diverses fonctions utiles telles que:

    Visqueux (ou non visqueux) analyse d’un profil d’aile existante, permettant
        transition forcée ou libre
        séparation transitoire bulles
        séparation limitée de bord de fuite
        portance et la traînée prédictions juste au-delà du Cz max
        Correction de compressibilité Karman-Tsien
        numéros fixes ou variables Reynolds et / ou Mach
    La conception et la reconception de Airfoil par modification interactive de la distribution de vitesse de surface, en deux modes:
        Procédé plein inverse, basée sur une formulation complexe de cartographie
        Mixte inverse méthode, une extension de la méthode de panneau de base de XFOIL
    Airfoil refonte de modification interactive des paramètres géométriques tels que
        épaisseur max et le carrossage, la position apogée
        LE rayon, épaisseur TE
        ligne de cambrure par spécification de la géométrie
        ligne de cambrure via spécification de changement de chargement
        rabat déviation
        géométrie de contour explicite (via curseur à l’écran)
    Le mélange des profils
    Écriture et la lecture de coordonnées de voilure et d’enregistrer des fichiers polaires
    Tracé de la géométrie, les distributions de pression, et de multiples polaires

Remarque: Le code source pour Xfoil lui-même est le même pour Unix et Win32. Le répertoire de la bibliothèque de la parcelle (de plotlib) a un sous-répertoire win32 séparé. Voir tous les fichiers README pour plus d’informations.

Remarques Win32: Interaction avec Win32 XFOIL est à travers une fenêtre de la console de texte type DOS. Certains de Win32 OS’es de Microsoft (Win95/98/ME) ont des limitations sur # de lignes dans une fenêtre de console et ne peut pas afficher entièrement menus ou sortie XFoil. Win95/98/ME ont également d’autres lacunes à l’égard de l’utilisation et de la stabilité des ressources bien XFOIL fonctionne sous ces OS’es. Windows NT, Win2000 et Windows XP sont les plates-formes Win32 recommandées.
Win32 Exe Notes: Les exécutables Win32 ont été compilées en utilisant le processeur Intel Fortran Compiler 5,01 à 15 et Visual C + 6,0. Le compilateur Intel (de grâce à Tom Clarkson à Intel) a été utilisé pour optimiser les exécutables pour P3 et P4 Pentium architectures. Les exécutables XFoil devraient fonctionner sur n’importe quelle machine Win32 Pentium que les options de compilation ont été utilisés pour inclure à la fois le code optimisé et Pentium ou AMD générique code du processeur pour la portabilité.
